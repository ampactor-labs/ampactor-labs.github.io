Selected Work

MENTL — Self-Hosting Compiler Mentl · WASM
A programming language that compiles itself.
Full Hindley-Milner type inference with row-polymorphic effect algebra; one mechanism (handle/resume) replaces exceptions, generators, async, DI, and mocking. L1 bootstrap produces bit-identical WASM across generations. Graph-native IR.
SONIDO — Cross-Platform DSP Kernel Rust · CLAP · no_std
Write audio code once; run it unchanged on a desktop plugin and a $20 microcontroller.
Three-layer kernel architecture: identical Rust on desktop CLAP plugins and bare-metal ARM (Cortex-M7 @ 480 MHz). 1,369 tests, 14 crates, zero-heap, zero-trait-dispatch. Firmware flashed to physical Daisy Seed hardware.
TOKENSAFE + SCRY — Solana Security Scanner TypeScript · Rust
A security product that charges real money to tell you whether a crypto token is a scam — shipped to production.
Nine deterministic on-chain checks; no ML, no third-party oracles. Paid via x402 micropayments ($0.008/request — payment is authentication). Web UI + Telegram bot; free MCP tool.
2-TOP — Rollback Netcode Brawler Rust · Bevy
Online multiplayer that stays perfectly in sync across phones — deterministic rollback netcode.
Provably deterministic cross-platform simulation, fixed-point math, replay codec with frame-accurate scrub, bevy_ggrs + Matchbox WebRTC. 275 tests; cross-platform determinism matrix in CI.
CLOB — Ternary Intelligence Kernel Rust
A ternary AI kernel that compiles what it learns into native machine code, on a laptop CPU.
Ternary recurrent kernel. The objective is energy: joules per nat, driven down. A crystallization loop turns prediction errors into compiled ternary modules. The compile path lowers a ternary matrix to native x86-64. Targets a laptop CPU, no GPU. In development.
TURBOSORT — SIMD Sort Library Rust · AVX2
A SIMD sort that beat std by about 3× on my laptop. Re-checking the number on better hardware.
All 10 primitive numeric types. AVX2 sorting networks for small arrays, LSD radix sort for large, runtime CPUID dispatch. No unsafe trait impls, no FFI, no nightly; full no_std.
BROWSORE — Browser Automation MCP TypeScript · Playwright
Lets an AI agent drive a real browser. 40 tools, and it respects robots.txt and rate limits.
Robots.txt compliance (RFC 9309), per-domain rate limiting, exponential backoff, prompt-injection detection. 40 MCP tools, CLI, programmatic API, and autonomous agent loop.
FLOWPILOT — Algorithmic Trading Engine Rust
A 13-crate Solana trading engine. The strategy can't beat institutional order flow, so it doesn't make money.
Discovers trending Solana tokens, filters through a 4-layer safety gate, executes via Jupiter with graduated take-profit and on-chain trailing stop-loss. 13-crate workspace; paper mode default.
